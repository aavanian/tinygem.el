** Quelpa or Straight.el

#+BEGIN_src elisp
  (use-package tinygem
    :quelpa (tinygem :fetcher github :repo "aavanian/tinygem.el"))
#+END_src

#+begin_src elisp
  (use-package tinygem
    :straight (:host github :repo "aavanian/tinygem.el" :branch "main"))
#+end_src

** Manual

Put =tinygem.el= in your =load-path=, and this in your init file:

#+BEGIN_src elisp
  (require 'tinygem)
#+END_src
